# Content Discipline

Read at Phase 2 (Plan) for content scopes and at Phase 3 (Standards) for any work producing copy — landing pages, articles, social posts, emails, AI-generated content.

| Standard | File | When |
|----------|------|------|
| Brand voice | [`brand-voice.md`](brand-voice.md) | Phase 3 every content scope |
| SEO | [`seo.md`](seo.md) | Phase 3 for any public-facing surface |
| Content strategy | [`content-strategy.md`](content-strategy.md) | Phase 2 — informs planning and editorial cadence |

---

## Companion documents (project-specific)

- **`BRAND_BOOK.md`** at the project root — carries the actual voice samples, banned phrases, tone scales, signature moves. These standards govern *how* to apply it; the brand book carries the specifics.
- **`EDITORIAL_CALENDAR.md`** — for projects with sustained content programmes.
- **`STYLE_GUIDE.md`** — for projects with house-style rules beyond voice (capitalisation, numerals, citation, dates).

---

## Verification checklist (Phase 5)

```
[ ] Brand voice review (against BRAND_BOOK.md scales + banned phrases)
[ ] SEO checklist green for public surfaces
[ ] Fact-check completed; sources cited where appropriate
[ ] Legal / compliance review where regulated content
[ ] Accessibility check for content surfaces (alt text, heading hierarchy, plain language)
[ ] AI-generated content explicitly reviewed (LLMs default to generic)
```
